   Why Have I logged these actions?

   I have logged the below 5 activities because in the real time I have observed that users mostly like to perform the below actions.

   Upvotes: The count of upvotes tells us what are all the answers liked by user for a given question.

   Downvotes: The count of downvotes tells us what are all the answers disliked by user for a given question.

   Answer marked as favourite(*): This tells how many answers are marked by an user as favourite.

   Mouse hover on Certified Answer(Green tick): This tells us whether user has read certified answer.

   Number of times user tried to copy text/code: This tells us whether user has liked a code/text and tried to copy it.


   Analysis and findings

    1) I have observed that one of the most common activity performed by a user is upvoting or downvoting an answer.

    2) I have observed most of the users wish to mouse hover only on the certified answer(having green tick).

    3) I have observed that if an user likes an answer he tries to copy the code and implement it on his machine.


   Explaining the patterns

    From the intercative visualization we can see that number of upvotes,downvotes and hovering on the certified answer count is more compared to others.Which clearly explains that the actions tracked above are the most frequently         performed actions by the users
